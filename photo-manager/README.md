# 📸 照片与媒体管理工具集 (Photo Manager Utilities)

本项目包含一系列使用 Python 编写的本地照片及视频管理工具。主要用于相册整理、小尺寸图片清理（优化备份限制）以及动态照片同名视频分类归档。

---

## 🛠️ 工具集介绍

### 1. 高性能照片批量管理器 (`photo_manager.pyw`)
这是一个高画质、多线程的本地照片批量浏览器与整理工具，支持 Windows 高 DPI 锐化。
* **特色功能**：
  * **高 DPI 支持**：自动开启 DPI-Aware 模式，防止系统缩放导致的界面或文字模糊。
  * **异步极速解码**：后台使用 4 线程 ThreadPoolExecutor 进行缩略图加速解码。
  * **LRU 缓存机制**：内置最大 150 张缩略图的缓存池，防抖滚轮加载，极速流畅。
  * **便捷操作**：
    * **滚轮**：上一页/下一页。
    * **Ctrl + `+`/`-`**：缩放网格尺寸。
    * **鼠标左键拖拽**：滑动批量选择照片。
    * **鼠标右键单击**：浮动显示 EXIF 详细元数据（包含分辨率、相机、GPS坐标等）。
    * **鼠标右键双击 / Delete 键**：快速将选中照片安全移入“最近删除”暂存文件夹。

### 2. 小尺寸图片自动清除归类工具 (`clean_small_photos.pyw`)
专为 Google Photos 备份优化设计的图片清理筛选工具。
* **特色功能**：
  * **流式读取**：仅读取图片头部数据（不进行全像素内存解码），秒级处理万级相册。
  * **规则筛选**：自动扫描所有子文件夹，将宽或高任一低于 `256` 像素的图片安全剪切移出，并保持原有的目录结构。
  * **防覆盖机制**：若目标目录存在同名文件，自动追加编号重命名。

### 3. 动态照片同名 MP4 整理工具 (`move_duplicate_mp4.pyw`)
用于整理动态照片（Live Photos）生成的大量同名短视频。
* **特色功能**：
  * **智能识别**：自动匹配并移动所有与 JPG/JPEG 静态图片同名的 MP4 视频。
  * **独立媒体保护**：不匹配或无同名图片的独立视频将安全保留在原位，仅归档“多余的”同名动态视频。
  * **结构保持**：递归扫描时保持原有的子文件夹层级目录，并在目标目录自动创建对应结构。

## 📦 环境依赖

工具基于 Python 3 运行，主要依赖 `tkinter` (标准库) 与 `Pillow` 图像处理库。
如果本地未安装 `Pillow`，`photo_manager.pyw` 与 `clean_small_photos.pyw` 会尝试在启动时**自动静默安装**。

### 手动安装依赖：
```bash
pip install Pillow
```

---

## 🚀 启动与使用

所有工具均以 `.pyw` 后缀命名，支持在 Windows 系统下直接双击无控制台窗口运行（GUI 可视化操作）。

1. 双击运行任一工具：
   - 运行主管理器：双击 `photo_manager.pyw`
   - 运行小图整理：双击 `clean_small_photos.pyw`
   - 运行同名视频归档：双击 `move_duplicate_mp4.pyw`
2. 在界面中点击 **“选择文件夹”** 设置对应的源目录与目标目录。
3. 点击启动按钮执行整理或浏览。
