# NodeSeek 自动任务助手 (发帖/回帖油猴脚本)

一个适用于 [NodeSeek](https://www.nodeseek.com/) 论坛的自动化任务助手。支持自动发布新讨论帖（如每日打包订阅热帖）、挂机自动回帖。具备自适应词云学习、多维度智能过滤屏蔽、跨页连续挂机、以及精致自适应控制面板配色。

---

## ✨ 核心特色

### 1. 自动发帖与回帖双模式支持
* **自动发帖 (Post Task)**：支持通过导入 JSON 任务文件，自动化发布新主题帖。可以直接将爬取到的日常订阅数据 (Markdown 格式) 一键封装并自动在指定板块 (如 `info` 情报板块) 发表。
* **自动回帖 (Reply Task)**：在后台创建隐藏的 `iframe` 沙盒加载帖子详情页，自动检测编辑器，模拟填充文本并触发原生提交。支持在 DOM 模拟失败时自动降级至 API 接口发包模式 (`/api/content/new-comment`)。

### 2. 智能学习与自适应权重系统
* **动态增重**：如果别人的回帖中频繁出现我们已有的某条预设回复，该回复将根据出现次数进行加分（`weight += 0.15 * 出现次数`，上限 `5.0`），被我们挑选的概率大幅上升。
* **动态衰退**：若某条预设回复一直未被其他人使用，算法会对扣分，防止机器化呆板的回复频繁出现。
* **贴内高频词云学习**：自动清洗分析贴内所有人回帖，提取高频词云并自动学习为临时关联词库。

### 3. 跨页挂机与排重机制
* **历史防重**：记录最近 5 次发送文本，防止刷版或被版主禁言。
* **自动挂机**：列表页无符合条件的帖子时，触发倒计时，支持自动识别并跳转下一页，继续安全恢复挂机执行。

## 📥 智能任务 JSON 格式

你可以通过控制面板的 **导入 JSON 任务** 导入待执行任务：

### 1. 自动发帖任务 (Post Task)
```json
[
  {
    "taskType": "post",
    "title": "🔥 NodeSeek 过去 24 小时热帖打包订阅 (2026-06-13)",
    "category": "info",
    "content": "这里是帖子的 Markdown 正文内容..."
  }
]
```

### 2. 自动回帖任务 (Reply Task)
```json
[
  {
    "taskType": "reply",
    "postId": "12345",
    "title": "帖子标题 A",
    "reply": "自定义回复文本"
  }
]
```

## 🛠️ 安装与使用

1. 安装 Chrome/Edge 的 **Tampermonkey** 浏览器插件。
2. 创建新脚本，将 `nodeseek_auto_tasks.user.js` 的完整代码复制并保存。
3. 登录并访问 [NodeSeek 首页](https://www.nodeseek.com/)。
4. 在控制面板导入 JSON 任务或配置参数后，点击 **开始自动任务**。
