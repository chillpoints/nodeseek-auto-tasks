# NodeSeek 自动回帖/水贴助手 (油猴脚本)

一个适用于 [NodeSeek](https://www.nodeseek.com/) 论坛的自动化加权挂机回帖助手。支持自适应词云学习、多维度智能过滤屏蔽、跨页连续挂机、以及精致自适应控制面板配色。

---

## ✨ 核心特色

### 1. 拟人与发包双通道机制 (Double-Channel Post)
* **通道一 (DOM 模拟优先)**：脚本在后台创建隐藏的 `iframe` 沙盒加载帖子详情页，自动检测编辑器（支持 CodeMirror 富文本和标准 Textarea），模拟填充文本并触发原生提交按钮。此模式可以完美继承浏览器的 Cookie、UA 和指纹，抗 Cloudflare 验证码性能极强，最符合人类正常浏览行为。
* **通道二 (API 发包备用)**：如果 `iframe` 受到同源限制、超时或 DOM 解析失败，脚本会自动降级至 API 接口发包模式，通过读取主页面 Meta 标签中的 `Csrf-Token`，向 `/api/content/new-comment` 直接发起 Ajax 提交，确保回帖高成功率。

### 2. 智能学习与自适应权重系统 (v1.3.0+)
* **动态增重**：如果别人的回帖中频繁出现我们已有的某条预设回复，该回复将根据出现次数进行加分（`weight += 0.15 * 出现次数`，上限 `5.0`），使其被我们挑选的概率大幅上升。
* **动态衰退**：如果我们的某条预设回复在扫描的数个帖子中几乎从未被其他人使用过，算法会对其执行扣分（`weight = Math.max(0.1, weight - 0.02)`，下限 `0.1`），防止机器化呆板的回复频繁出现。
* **贴内高频词云学习**：后台自动清洗分析帖内前人所有回帖，若检测到某条非预设回复重复出现 **>= 3 次**，便自动习得。通过内置轻量正则分词与 `STOPWORDS` 中文停用词库，提取贴子标题的词云设为关键词与该新回复自动关联并回写预设库。

### 3. 近 5 次历史回帖跨页排重
* 通过 `ns_auto_reply_recent_replies`（GM 存储）追踪和记忆您最近 5 次的发送文本，跨页跳转不丢失。
* 采用漏斗退避去重算法，当匹配组过滤后为空时，自动向 default 组或放宽排重限制，保证高频发帖中即使备选集极小也绝不与上一贴重复。
* 所有的回帖内容**绝对 100% 必须来源于当前的预设回复库**（包括用户手写的或自动学习到的最新词库），不允许脱离预设范围。
* **AI 语义挑选器**：AI 模块被重构为“智能语义分析挑选器”。启用 AI 时，AI 模块分析帖子标题后，从备选集里语义挑选出最贴切的一项原文原样返回，既保障了 100% 预设安全，又保留了 AI 的智能上下文分词挑选能力。

### 4. 连续跨页自动挂机 (v1.4.0+)
* 当当前帖子列表内所有符合过滤要求的帖子都已被完全回复时，触发 3 秒倒计时，在状态栏即时提醒 `即将翻页 (3s)`。
* 优先 DOM 查找下一页链接，若未找到则根据当前 URL 结构进行智能算术推导（如 `/page-x` 递增为 `/page-x+1`）。
* 在倒计时结束跳转前向 GM 写入全局运行标识，新页面加载后将在 1.5 秒缓冲渲染后**自动模拟点击面板的“开始自动回帖”按钮**，无缝恢复挂机，实现完全脱手的自动跨页回帖。

## 🎨 配色定制与重置
* 控制面板采用 Premium UI 复古暖黄风格（默认配色为：Background `#FFFDF5`, Foreground `#101010`, Accent `#CB4B16`）。
* 面板整合了紧凑的 RGB 调色盘，您可以自行定制面板的背景色、前景色与主题高亮色。
* 配色区域附带 **重置** 按钮，若调色发生极端冲突，可一键秒级恢复默认经典暖米黄配色。

## 🛠️ 安装与使用

1. 安装 Chrome/Edge 的 **Tampermonkey** 浏览器插件。
2. 创建新脚本，将 `nodeseek_auto_reply.user.js` 的完整代码复制并保存。
3. 登录并访问 [NodeSeek 首页](https://www.nodeseek.com/)。
4. 在页面右上方悬浮的控制面板上进行您的参数个性化定制后，点击 **开始自动回帖**。

> [!IMPORTANT]
> **安全防封建议**
> 1. 请不要将回帖间隔设置得过于激进（建议不低于 30-60 秒）。
> 2. 单次运行上限不宜设置过高，建议配合日常互动每日手动运行。
> 3. 初次使用时，请打开「运行日志」选项卡观察一两次回帖行为，确信正常后再最小化挂机。
